## 事前準備

### 1. Node.js のインストール

[mise](https://mise.jdx.dev/) を使用して Node.js をインストールする。

```bash
mise install
```

### 2. ビルド

```bash
npm install
npm run build
```

## 実行方法

```bash
# ヘルプを表示
node dist/index.js --help

# 基本的な使用方法
node dist/index.js
```

