# Plan Output Directory

- Save all plan and design documents to `tmp/plans/` instead of `docs/plans/`
- This applies to the `superpowers:writing-plans` and `superpowers:brainstorming` skills
- Format: `tmp/plans/YYYY-MM-DD-<topic>.md`
- The `tmp/` directory is gitignored — plans are ephemeral working documents, not permanent artifacts
